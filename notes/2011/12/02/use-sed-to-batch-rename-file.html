Sed批量去拓展名

|- dev.gb.conll06.raw
|- test.gb.conll06.raw
|- train.gb.conll06.raw

想要去掉其中的后缀.raw，用个小脚本就可以搞定：

for file in `ls`; do mv $file `echo $file | sed 's/\.raw$//'`; done
