像'printf'一样打印变长参数

动机

我想实现一个可以像printf一样带参数模板的log打印函数，在调用

{% highlight c %} linecount = 1000; writelog(“process %d lines”, line_count); {% endhighlight %}

打印出

process 1000 lines

方法

实现方法是使用stdarg.h中提供的变长参数接口。vprintf、vsprintf、vfprintf等等，我们可以用如下的方法实现write_log函数：

{% highlight c linenos %} void writelog(int lvl, char *fmt, …){ … valist list; vastart(list, fmt); vfprintf(stderr, fmt, list); vaend(list); … } {% endhighlight %}
